# Frontend

React + Vite + TypeScript scaffold.

## Setup

```bash
cp .env.example .env
npm install
npm run dev      
```

## Build

```bash
npm run build
```

## Routes

| Path            | Page               |
|-----------------|--------------------|
| `/`             | Home               |
| `/login`        | Login              |
| `/register`     | Register           |
| `/startups/:id` | StartupView        |
| `/dashboard`    | InvestorDashboard  |
| `/messages`     | Inbox              |
